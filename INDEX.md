## How to use this repo in 60 seconds
1. Read `00_ADMIN/CONTEXT_PACK.md` for current assumptions.
2. Open `00_ADMIN/RUN_LOOP.md` and follow the per-topic checklist.
3. Start with `01_RESEARCH_PROMPTS/01_audience_psychology.md`.
4. Paste ChatGPT output into matching file in `02_RESEARCH_OUTPUTS/`.
5. Update `00_ADMIN/CONTEXT_PACK.md`, `00_ADMIN/DECISIONS.md`, and `00_ADMIN/STATUS.md`.
